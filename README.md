# Password_Strength_Analyzer

A Python-based security tool for analyzing password strength and generating custom wordlists for security testing.

## Features

###  Password Strength Analyzer
- Checks password strength on a scale of 0-4
- Calculates password entropy
- Estimates crack time
- Provides improvement suggestions
- Analyzes character types used

###  Custom Wordlist Generator
- Generates wordlists from personal information
- Applies leet speak transformations
- Adds common suffixes and patterns
- Includes year variations
- Exports to text files for security testing

## Installation

1. Clone this repository:

          git clone https://github.com/yourusername/password-security-tool.git
          cd password-security-tool
2. Run the tool (no external dependencies required):
   
         python main.py
## Usage
Interactive Mode

      python main.py
### Then choose from the menu:

Option 1: Check Password Strength

Option 2: Generate Custom Wordlist

Option 3: Exit
